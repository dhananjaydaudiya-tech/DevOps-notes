What is Kubernetes?
Kubernetes is a container orchestration platform used to manage Docker containers.
Kubernetes Components
Pod
Smallest deployable unit in Kubernetes.
Node
Machine where pods run.
Cluster
Group of nodes.
Deployment
Used to manage application deployment.
Service
Used to expose applications.
Kubernetes Commands
kubectl get pods
Shows running pods.
kubectl get nodes
Shows cluster nodes.
kubectl create deployment
Creates deployment.
kubectl delete pod
Deletes pod.
Advantages of Kubernetes
Auto Scaling
Load Balancing
High Availability
Self Healing
