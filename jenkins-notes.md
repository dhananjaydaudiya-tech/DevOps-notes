What is Jenkins?
Jenkins is an automation tool used for CI/CD pipelines.
Features of Jenkins
Continuous Integration
Continuous Deployment
Automation
Plugin Support
Jenkins Commands
Start Jenkins
systemctl start jenkins
Stop Jenkins
systemctl stop jenkins
Check Jenkins Status
systemctl status jenkins
Advantages of Jenkins
Open Source
Easy Automation
Supports Multiple Plugins
Fast Deployment
CI/CD Workflow
Code -> Build -> Test -> Deploy
