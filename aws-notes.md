What is AWS?
AWS (Amazon Web Services) is a cloud computing platform provided by Amazon.
AWS Services

Used to create virtual servers.
S3
Used for cloud storage.
IAM
Used for user access and permissions.
VPC
Used for network configuration.
RDS
Used for managed databases.
