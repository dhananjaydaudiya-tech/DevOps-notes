Basic Linux Commands
pwd
Used to check current directory.
ls
Used to list files and folders.
cd
Used to change directory.
mkdir
Used to create a new folder.
touch
Used to create a new file.
rm
Used to remove files.
cp
Used to copy files.
mv
Used to move or rename files.
clear
Used to clear terminal screen.
File Permission Commands
chmod
Used to change file permissions.
chown
Used to change file ownership.
Process Commands
ps
Displays running processes.
top
Shows system processes in real time.
kill
Used to stop a process.
Network Commands
ping
Checks network connectivity.
ifconfig
Displays IP address information.
netstat
Displays network connections.
