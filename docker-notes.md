What is Docker?
Docker is a containerization platform used to build, run, and deploy applications.
Docker Commands
docker --version
Check Docker version.
docker pull
Download Docker image.
docker run
Run Docker container.
docker ps
Show running containers.
docker images
List Docker images.
docker stop
Stop container.
docker rm
Remove container.
docker rmi
Remove Docker image.
Advantages of Docker
Lightweight
Fast deployment
Easy scalability
Works on multiple environments
